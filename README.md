# Python Intro Exercises

---

## Exercise 1

**Problem:**

Find the lesser of two numbers.

**Example:**

| Input | Output |
|-------|--------|
| 5, 10 | 5      |
| 15, 5 | 5      |
| 7, 7  | 7      |
| 100, 99 | 99    |

## Exercise 2

**Problem:**

Find the sum from 3 numbers that are not divisible by 11.

**Example:**

| Input            | Output |
|------------------|--------|
| 11, 22, 33, 44 | 0      |
| 1, 2, 3, 4, 5     | 15     |
| 10, 20, 30, 40    | 100    |
|11, 12, 13, 14    | 39     |

## Exercise 4

**Problem:**

Check if a number is greater than 10 and return "YES" or "NO".

**Example:**

| Input | Output |
|-------|--------|
| 5     | NO     |
| 10    | NO     |
| 11    | YES    |
| 20    | YES    |
